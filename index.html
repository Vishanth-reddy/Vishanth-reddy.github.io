Future‑Aware Distillation GNN for Daily Stock Trends

Author: Vishanth Reddy Bijjur

This project builds daily correlation graphs from returns, trains a teacher GNN with a limited future window, distills it into a student GNN that only sees the past, and uses a Crew‑style decision agent to emit BUY/SELL/HOLD. I implemented the data/graph pipeline, teacher–student models with knowledge distillation, JSON inference, Crew AI decisioning, and a Backtrader evaluation harness.

Repo

Branch: bijjur‑sprint4

Key modules: src/UI/gnn.py (pipeline + agents) and src/Backtesting/gnn_backtest.py (backtest).

How it works

  1. Graph construction: compute N‑day rolling Pearson correlations over returns to form daily adjacency matrices.
  2. Teacher: a GCN→GRU classifier sees past + small future (t+Δ) to learn next‑day up/down logits.
  3. Student: same architecture but no future inputs; trained with KD loss (KL to teacher logits + CE to labels).
  4. Inference: with today’s graph + feature window, output [{stock, student_prob}].
  5. Decisioning: Crew agent converts student_prob into {"recommendation"}.
  6. Backtesting: use daily recommendations in Backtrader to estimate Sharpe, returns, and drawdown.

Agents

  • PriceLoaderAgent — fetch OHLC, compute returns
  • GraphBuilderAgent — rolling correlations → adjacency
  • TeacherTrainAgent — temporal GNN with t+Δ
  • StudentDistillAgent — KD(teacher, student) + CE
  • InferenceAgent — emit {stock, student_prob}
  • DecisionAgent — BUY/SELL/HOLD from probability

User Story (DishFT‑GNN)

Acceptance: daily adjacency tensors from rolling correlations; teacher converges on next‑day labels; student matches teacher logits and predicts labels; inference returns JSON; Crew agent outputs BUY/SELL/HOLD. Conditions: validated shapes/thresholds; clean training; robust to gaps/low‑vol; decisions consistently valid. DoD: loader, graph builder, teacher/student, inference JSON, Crew agent, and end‑to‑end test in place.

Show detailed criteria
  • Graph Construction: rolling N‑day correlations → adjacency per date.
  • Teacher Training: past+future (t+Δ) embeddings, next‑day labels.
  • Student Distillation: KL(student‖teacher) + classification loss.
  • Inference Output: JSON schema [{stock, student_prob}].
  • Crew AI Decisioning: returns {"recommendation":"BUY|SELL|HOLD"}.
